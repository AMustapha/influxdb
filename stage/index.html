This is the docker-compose stack I use for my home network’s monitoring. It contains the following containers:

  • InfluxDB to store all of the metrics (the central piece of the setup).
  • Grafana to show pretty graphs of all the metrics
  • Kapacitor to email alerts when things break.
  • Raspberry Pi or other Linux system for external monitoring.

I also use the following services:

  • SparkPost free tier for sending alerts emails.
  • Tor for our external Linux system to get around NAT/Firewalls.

This website is my brain dump on how I got monitoring and alerting setup on my home server from scratch. If you plan on using it for your server you’ll need to be on the lookout for my hostnames and IP addresses and substitute them with your own.
